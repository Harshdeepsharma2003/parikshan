**Pariksan** is a web-based application designed to facilitate online exams, quizzes, and assessments. It allows users to securely register, attempt tests, and view results, while enabling admins to create, manage, and monitor exams efficiently.

## Features

- **User Registration & Login:** Secure authentication system for students and admins.
- **Exam Management:** Admins can create, edit, and schedule multiple-choice or subjective exams.
- **Real-time Test Taking:** Students can attempt exams online with auto-timed sessions.
- **Result & Analytics:** Instant scoring, detailed performance reports, and progress tracking.
- **Profile Management:** Users can update their personal information and view past results.

## Technologies Used

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Spring Boot (Java)
- **Database:** MySQL
- **Build Tools:** Maven / Spring Tool Suite (STS)
- **Others:** JWT Authentication, RESTful APIs
